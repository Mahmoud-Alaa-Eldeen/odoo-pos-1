Install Restrict User to Default POS Module

Installation in just few seconds.

After purchase and download the module to your computer, you have to simply click "install" button.

Note: Do not forget to copy the module to the customized addon folder. Avoid put this module with other odoo module's.

Create and Configure a new Point of Sale

It´s important to have more than one POS created and Configured.

To notice the changes that will be made by this module, it´s important to have at least two point of sale created and configured.

If you want to create a new POS, please click in Point of sale menu. 1. Go to vertical menu "Configuration -> Point of Sale" 2. Click button "Create"

Restrict user to his Default Point of Sale

Assign a POS to the user.

At this point, you have all the condition to set a POS to his user.

Note: Do not forget that the POS can be assign to more than one user. Not at the same time, but one by one. It means that two users can be assigned to the same POS, but it´s forbidden to acess it at the same time.

The User dashboard

the user view will be limited to his default pos.

The Manager "Administrator" dashboard

the Manager view will be abble to see all pos created.
